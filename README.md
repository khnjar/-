# 📘 ICM Core Engine v1 – نظام القرار الذكي

## 🎯 ما هذا النظام؟

**ICM Core Engine** هو نظام قرار ذكي متعدد الوكلاء يحلل البيانات من 4 زوايا مختلفة:

```
┌─────────────────────────────────────┐
│   البيانات المدخلة (Input Data)     │
│  ضغط، حرارة، نشاط، إجهاد           │
└─────────────────────────────────────┘
              ⬇️
┌─────────────────────────────────────┐
│    4 وكلاء ذكيين يحللون البيانات    │
│  ❤️ صحة | 🌡️ بيئة | 💰 اقتصاد | ⚠️ مخاطر │
└─────────────────────────────────────┘
              ⬇️
┌─────────────────────────────────────┐
│    محرك ICM (الدمج والتركيب)        │
│    يجمع آراء الوكلاء في قرار واحد   │
└─────────────────────────────────────┘
              ⬇️
┌─────────────────────────────────────┐
│      القرار النهائي الذكي           │
│  (مع شرح تفصيلي للأساس المنطقي)    │
└─────────────────────────────────────┘
```

---

## ✨ الميزات الرئيسية

✅ **تحليل فوري** - نتائج لحظية عند تعديل البيانات
✅ **4 وكلاء ذكيين** - كل وكيل متخصص في مجال
✅ **رسوم بيانية** - تتبع التطور عبر الوقت
✅ **ذاكرة الحكمة** - يحفظ آخر 5 قرارات
✅ **آمن تماماً** - بيانات محلية، لا تُرسل للخادم
✅ **واجهة جميلة** - RTL عربية كاملة

## 🎮 كيفية الاستخدام

### الخطوة 1: اعدّل المنزلقات
```
ضغط الدم:     80-200 mmHg (الطبيعي: 120)
الحرارة:       20-45°C (المريح: 28)
النشاط:        0-100 (متوسط: 50)
الإجهاد:       0-10 (منخفض: 3)
```

### الخطوة 2: شغّل المحاكاة
```
اضغط الزر الأزرق الكبير: "تشغيل المحاكاة (ICM Core)"
```

### الخطوة 3: اقرأ النتائج
```
- شاهد آراء الـ 4 وكلاء على اليمين
- اقرأ القرار النهائي في الوسط
- لاحظ درجة الخطر الإجمالية
```

### الخطوة 4: تتبع التطور
```
- انظر إلى ذاكرة الحكمة (آخر 5 قرارات)
- اتبع الرسم البياني (آخر 7 محاولات)
```

## 🤖 الوكلاء الأربعة

### ❤️ وكيل الصحة
يحلل: ضغط الدم، الإجهاد، النشاط البدني

| الحالة | الوصف |
|-------|--------|
| 0-30% | ✅ آمن تماماً |
| 30-60% | ⚠️ تحذير |
| 60%+ | 🔴 خطير جداً |

**النصائح:**
- ضغط > 160 mmHg = طارئة طبية
- إجهاد > 7 = خذ فترة راحة
- نشاط > 80 مع حرارة عالية = خطير

---

### 🌡️ وكيل البيئة
يحلل: درجة حرارة الجو فقط

| الحرارة | الحالة |
|--------|--------|
| 20-25°C | 🟢 بارد لطيف |
| 25-35°C | 🟡 دافئ |
| 35-40°C | 🔴 حار جداً |
| 40+°C | 🟣 خطر شديد |

---

### 💰 وكيل الاقتصاد
يتوقع: التكاليف الصحية المستقبلية

**الحساب:**
```
نشاط عالي + إجهاد مرتفع = تكاليف عالية محتملة
نشاط منخفض + إجهاد منخفض = تكاليف منخفضة
```

---

### ⚠️ وكيل المخاطر
يجمع: جميع العوامل في درجة خطر واحدة

**المعادلة:**
```
خطر = (ضغط/200 × 0.4) + (حرارة/50 × 0.2) 
      + (نشاط/100 × 0.2) + (إجهاد/10 × 0.2)
```

## 📊 تفسير النتائج

### الألوان الثلاث

```
🟢 أخضر (0-40%)
   ✅ حالة مستقرة – متابعة روتينية
   → يمكنك الاستمرار بنشاطك الطبيعي

🟡 أصفر (40-65%)
   ⚠️ حالة تستدعي المراقبة والوعي
   → انتبه، لكن لا تقلق كثيراً

🔴 أحمر (65-100%)
   🚨 تدخل فوري مطلوب
   → خذ إجراءات فوراً
```

## 💾 ذاكرة الحكمة

تحفظ آخر 5 قرارات:

**مثال:**
```
14:35  خطر 48%  ⚠️ حالة تستدعي المراقبة
14:30  خطر 42%  ⚠️ حالة تستدعي المراقبة
14:25  خطر 35%  ✅ حالة مستقرة
14:20  خطر 50%  ⚠️ حالة تستدعي المراقبة
14:15  خطر 38%  ✅ حالة مستقرة
```

**الفائدة:** تتبع التطور والأنماط
